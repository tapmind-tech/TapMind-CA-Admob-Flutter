# TapMind Ads Flutter

A Flutter plugin for integrating TapMind ads into your Flutter applications. This plugin provides a unified interface for displaying ads on both Android and iOS platforms.

## Documentation
- [TapMind Android Integration](https://tapminds.gitbook.io/custom-adapter-sdk-1/custom-adapter-integration/android)
- [TapMind iOS Integration](https://tapminds.gitbook.io/custom-adapter-sdk-1/custom-adapter-integration/ios)

### 3. Platform Setup

**Android:**
- Minimum SDK: 24
- Add required permissions to `AndroidManifest.xml`

**iOS:**
- Minimum iOS: 15.0
- Run `pod install` in the `ios` directory


## Requirements

- Flutter SDK: >=3.3.0
- Dart SDK: ^3.9.2
- Android: minSdkVersion 24
- iOS: 15.0 or higher
